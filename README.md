Modern Frontend Developer Portfolio
A high-tech, "neon noir" inspired portfolio website showcasing modern web development techniques, custom animations, and responsive design.

Live Demo: https://my-portfolio-site41.netlify.app/

🚀 Overview
This project serves as my professional digital storefront. It features a sleek dark-themed interface with high-end aesthetic presentation, specifically designed to demonstrate proficiency in modern frontend frameworks and interactive UI/UX design.

Key Features:
Neon Noir Aesthetic: A consistent, high-tech visual theme.

Custom Animations: Including unique orbit animations and scroll-based reveal effects.

Fully Responsive: Optimized for all screen sizes, from mobile devices to full-screen desktop displays.

Modern Navigation: intuitive layout featuring Home, About, Skills, and Contact sections.

🛠️ Tech Stack
Core: React.js

Build Tool: Vite

Styling: Modern CSS / Neon-noir UI

Deployment: Netlify

Tools: Windsurf AI, Git/GitHub

📁 Project Structure
E-commerce Showcase: Features high-end branding and commercial web layouts.

SaaS Dashboard: Demonstrates data visualization and complex UI management.

Skills Section: Highlights technical proficiency in the React ecosystem.

⚙️ Local Development
Clone the repository:

Bash
git clone https://github.com/zolla-austin/portfolio.git
Install dependencies:

Bash
npm install
Run the development server:

Bash
npm run dev
